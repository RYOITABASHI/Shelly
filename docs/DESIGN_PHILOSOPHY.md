# Shelly 設計思想

## Shellyとは何か

Shellyは「自然言語 → 実行」のパイプラインをAndroid1台で完結させるアプリ。

- **チャットアプリ（ChatGPT等）** → 会話はできるが、実行できない。コードを生成しても、それをどこに置いてどう動かすかはユーザー任せ。
- **CLI（Claude Code等）** → 実行力は最強だが、ターミナルを使える人しか恩恵を受けられない。
- **Shelly** → チャットで話しかけるだけで、裏でCLIが実行する。ユーザーは「アプリ作って」と言うだけで、AIがファイル生成・git管理・テスト実行まで全部やる。その過程もチャットUIで見える。

チャットアプリにはない実行力、CLIにはない手軽さ。その間を埋める唯一のポジション。

## ターゲットユーザー

ChatGPTやGemini、Claudeのチャットアプリで満足しているユーザーは、そもそもShellyを使わない。Shellyのユーザーは：

1. **CLIを使ってものづくりをしたいけれど、CLIを使いこなしきれない人たち**
2. **チャットベースで開発を進めたい人たち**
3. **ゴリゴリ開発もできるが、エンジニアではないから専門用語やプログラムに詳しくないという人たち**

つまり、チャットとターミナルをハイブリッドで扱えるツール。

## コアアーキテクチャ: 「チャットUIの裏でCLIが走る」

Shellyはチャットインターフェースだが、裏ではCLI（Claude Code, Gemini CLI, Codex等）が実際にコマンドを実行している。これがShellyの味噌。

一見、CLIを使って自然言語のチャットだけをするのは贅沢で手間に見えるが、実はそれこそがShellyの最大の差別化ポイント。

### 2つのモード

- **Chatタブ（初心者向け）**: チャットUIで対話。CLIのパーミッションプロンプトはローカルLLMが翻訳し、ボタンで承認。ユーザーはTermuxの存在すら意識しない。
- **Terminalタブ（上級者向け）**: 生のターミナル。CLIを直接操作できる。

ユーザーの熟練度に応じて、どちらのタブを主に使うかが自然に分かれる。

## Termuxless設計

全くの初学者が、ShellyはもちろんTermux自体も使ったことがない状態でインストールしても、Shellyの全機能を満足に使えるようにする。

### 5層アーキテクチャ

```
Chat UI（自然言語入力）
    ↓
Intent Router（ローカルLLMがタスクを分類）
    ↓
Tool Orchestrator（CLI管理・パーミッション翻訳）
    ↓
Environment Manager（ツール自動インストール・認証）
    ↓
Termux Bridge（コマンド実行）
```

### インテントルーティング

ユーザーは「何を使うべきか」すら知らなくていい。

例: 「ポートフォリオ作って」
1. ローカルLLMが意図を解析 → 「Claude Codeが最適」と判断
2. Claude Code未インストールを検知
3. チャットで案内: 「このプロジェクトはClaude Codeで開発するのが最適です。現在インストールされていないので、セットアップを開始しますか？」
4. ユーザーが承認 → 自動インストール → 認証フロー → そのまま開発開始

ツール選定もセットアップも、全部チャットの中で完結する。

### CLIパーミッション翻訳

Claude CodeやGemini CLIが「このファイルを読んでいいですか？(Y/n)」と英語で聞いてくる。初心者はこれが理解できない。

Shellyでは：
- ローカルLLMがCLIの質問を初心者向け日本語に翻訳
- 「実行する」「実行しない」「テキスト入力」のボタンUIで提示
- 自動承認レベルも設定可能（全手動 / 読み取りのみ自動 / 全自動）

Gemini CLIは自然言語で質問してくるので、そのまま翻訳・表示すればよい。

## システムプロンプト自動注入

ローカルLLMをONにした時点で、Shellyの設計思想・利用可能ツール・プロジェクト情報が自動的にLLMのシステムプロンプトに注入される。ユーザーは何も設定する必要がない。

さらに、Settings画面でユーザーが独自のMDコンテキスト（設計ルール、コーディング規約等）を追加できる。

## まとめ

Shellyは「チャットアプリの手軽さ」と「CLIの実行力」を融合したハイブリッドツール。ユーザーがやりたいことを自然言語で伝えるだけで、最適なツール選定・環境構築・実行・結果表示まで、全てがチャットUIの中で完結する。
